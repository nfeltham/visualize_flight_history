# visualize_flight_history
Visualize flight data on a map. 

Easiest to run by first creating an anaconda env:

In the directory, run:

conda env create -f environment.yml

Then, configure the flight_data.csv file with your flight segments. If you only want to plot one segment, you can also just use single_trip_plot.
Run main.py, and your output video will be saved once the program is complete.
